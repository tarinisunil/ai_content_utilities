# Practical AI Utilities for Content

Small, practical AI tools for content engineering workflows.

This repository contains experiments and utilities focused on using LLMs to solve real-world problems in content systems—without heavy infrastructure or expensive setups.

## 🎯 Goals

- Build simple, useful AI utilities (not demos)
- Work with unstructured → structured content
- Explore lightweight RAG pipelines
- Improve content quality with AI-assisted tools
- Use mostly open-source and low-cost solutions

## 🧰 Tech Stack

- Python
- Ollama (local LLMs)
- FAISS / ChromaDB
- sentence-transformers
- Hugging face
- Open Router

## 🚀 What You'll Find

- Content extraction tools (Markdown/HTML → JSON)
- Schema validation with LLMs
- Lightweight RAG implementations
- Small CLI-based utilities
